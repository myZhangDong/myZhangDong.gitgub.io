计算机中的二进制

进位计数制中的概念

  • 基数 R

R代表基本数码的个数，二进制就是0和1这两个数。R进制的特点是逢R进1。基数R的数制称为R进制数。

  • 位权 Wi

权位Wi是指第i位上的数码的权重值，权位与数码所处的位置i有关，例如R=10(十进制数),各个数码的权为10^i^, i表示数码所处的位置。
个位i=0，位权就是10^0^=1; 十位i=1，位权是10^1^=10.
例如十进制：11.11 = 1×10^1^ + 1×10^0^ + 1×10^-1^ + 1×10^-2^
|R|进制|数码符号|进制规则|位权
|-|-|-|-|-|
|R=2|二进制数（2）|0、1|逢2进1|2^i^
|R=10|十进制数（10）|0~9|逢10进1|10^i^

二进制与十进制之间的转换

  • 二进制转十进制
    方法：按权展开，加权求和。
    例如：二进制数11.11转为十进制 = 1×2^1^ + 1×2^0^ + 1×2^-1^ + 1×2^-2^ = 2 + 1 + 0.5 + 0.25 = 3.75
  • 十进制转二进制
    整数部分：除2取余，直到商为0，最先得到的余数是最低位，最后得到的余数是最高位。
    小数部分：乘2取整，直到积为0或达到精度要求为止，最先得到的整数就是高位。
